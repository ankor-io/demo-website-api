Ankor Website API Demo - Integration via iFrame

Integration: via iFrame

  • Any "Share" link created from any of the Ankor Products can be embedded in your website via an iFrame.
  • This mechanism, while easy to implement, it provides minimal SEO benefit. It is suggested to ensure the pages and copy around this iFrame are SEO optimized.
  • The example below embeds a Presentation from Ankor Trips.
  • How the iFrame is rendered in this page is customizable; see MDN: <iframe>: The Inline Frame element .
