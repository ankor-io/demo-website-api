Ankor Website API Demo - Navigation by Geo

Hierarchical Geographic Navigation

  • This example uses geographic area names (the 'category' field) from the Ankor Website API response and maps these fields to the indexed record within the search provider.
  • Records indexed to our search provider in a format which allows for the UI to construct a hierarchical menu; similar to the way a set of e-commerce product categories would be used to filter products.
  • See this Algolia tutorial for more information on how a hierarchical menu can be used.
  • If filtering by using the dates, use dates within 2024.
