Ankor Website API Demo

Facet Destination Search

  • This example uses geographic area names (the 'category' field) from the Ankor Website API response and maps these fields to the indexed record within the search provider.
  • These (three level) geographic area names are then used as a facet to filter the yachts based on their location. This list can be formatted as a three-tiered dropdown if required by tweaking the record indexing and faceting.
  • If filtering by using the dates, use dates within 2024.
